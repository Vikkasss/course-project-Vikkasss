# Non-Functional Requirements для сервиса

| ID | Название | Описание | Метрика/Порог | Проверка | Компонент | Приоритет |
|----|----------|-----------|---------------|----------|-----------|-----------|
| NFR-01 | Защита пользовательских заметок | Шифрование личных цитат пользователей | AES-256 для текста заметок | Тесты шифрования | storage | High |
| NFR-02 | Производительность поиска | Быстрый поиск по цитатам и авторам | p95 ≤ 100ms при поиске | Нагрузочное тестирование | search | High |
| NFR-03 | Доступность сервиса | Сервис доступен для чтения заметок | SLA ≥ 99.5% | Мониторинг uptime | api | High |
| NFR-04 | Надёжность хранения | Сохранённые цитаты не теряются | 99.9% durability данных | Backup тесты | database | High |
| NFR-05 | Защита от спама и злоупотреблений | Ограничение создания/поиска цитат | ≤ 10 новых цитат/минуту, ≤ 60 поисков/минуту | Rate limiting | api | Medium |
| NFR-06 | Безопасность аутентификации | Защита аккаунтов пользователей | Argon2id для паролей | Security audit | auth | High |
| NFR-07 | Производительность загрузки | Быстрая загрузка списка цитат | p95 ≤ 200ms при 50 RPS | Нагрузочный тест | ui | Medium |
| NFR-08 | Резервное копирование | Регулярное сохранение данных | Ежедневные бэкапы | Проверка восстановления | backup | Medium |
