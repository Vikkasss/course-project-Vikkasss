# BDD сценарии для сервиса цитат

## Feature: Быстрый поиск цитат

### Scenario: Защита от массового добавления цитат
Given пользователь имеет лимит 10 цитат в минуту

When пользователь пытается добавить 15 цитат за 30 секунд

Then первые 10 цитат сохраняются успешно

And остальные 5 запросов получают ошибку "Слишком много запросов"

## Feature: Защита личных заметок

### Scenario: Шифрование пользовательских заметок
Given пользователь создаёт личную заметку

When заметка сохраняется в системе

Then текст заметки шифруется AES-256

And только пользователь может её прочитать

## Feature: Доступность сервиса

### Scenario: Сервис доступен в пиковые часы
Given вечернее время (пиковая нагрузка)

When 1000 пользователей одновременно открывают приложение

Then сервис отвечает на запросы

And время ответа ≤ 500ms
